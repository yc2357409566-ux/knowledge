# 第一个月：2026年3月 - A1初级：发音与基础（120小时目标）

> **阶段目标**：掌握法语发音系统、学会自我介绍、建立学习习惯
>
> **时间**：2026年3月21日 - 2026年4月20日
>
> **总学时**：约120小时（平均每天3-4小时）
>
> **关键里程碑**：
> - 第1周末：完全掌握法语字母和发音规则
> - 第2周末：能进行基本问候和自我介绍
> - 第3周末：掌握200个日常词汇
> - 第4周末：能进行2-3分钟的简单对话

## 🎯 周度目标

### 第1周（3月21-27日）：发音突破 ⭐
**目标小时**：30小时（周一至周日，平均4小时/天）

**核心任务**：
1. ✅ 学习26个法语字母及其发音
2. ✅ 掌握主要鼻化音 (nasales)：an, in, on, un
3. ✅ 学习特殊字符和发音规则（ç, è, é等）
4. ✅ 学习基本语调和重音规则
5. ✅ 记住30个最常用问候词汇

**每日时间分配**（4小时）：
- 早（40分钟）：字母发音反复练习 + Forvo听原音
- 中（60分钟）：鼻化音练习 + 舌位视频教学
- 晚（50分钟）：阅读字母词汇并跟读
- 加强（30分钟）：Anki卡片温习

**学习资源**：
- YouTube: "法语发音完全教程" (Linguoni)
- Pimsleur法语第1-5课
- Forvo发音示范
- RFI Savoirs第1-10课

**学习成果检查**：
- [ ] 能正确发音26个字母
- [ ] 能发音鼻化音
- [ ] 能跟读并理解30个常用词汇
- [ ] 能进行简单问候（Bonjour, Au revoir等）

### 第2周（3月28-4月3日）：基础问候与自我介绍 ⭐⭐
**目标小时**：30小时

**核心任务**：
1. ✅ 学习"是"动词（être）的现在时变位
2. ✅ 学习"有"动词（avoir）的现在时变位
3. ✅ 学习自我介绍的关键句型
4. ✅ 学习100个日常词汇（人物、职业、国家）
5. ✅ 能进行30秒自我介绍

**关键句型**：
```
Je suis [名字]        - 我叫...
Je suis [国籍]        - 我是...人
Je suis [职业]        - 我是...
J'ai [年龄] ans       - 我...岁
J'ai [特征]          - 我有...
```

**每日时间分配**（4小时）：
- 早（40分钟）：动词变位学习 + Anki练习
- 中（60分钟）：自我介绍句型练习
- 晚（50分钟）：看视频示范并模仿
- 加强（30分钟）：与自己进行对话

**学习资源**：
- Pimsleur第6-15课
- YouTube: "法语être和avoir动词"
- Assimil教材第1-5课
- 标准自我介绍模板

**学习成果检查**：
- [ ] 掌握être和avoir的6个变位
- [ ] 能进行30秒自我介绍
- [ ] 认识并能发音100个词汇
- [ ] 能理解他人的简单自我介绍

**核心任务**：

- TV5Monde初级课程（第1-7课）

## 💪 第一个月学习技巧

### 1. 建立日常习惯
- 每天同一时间学习
- 早餐后30分钟Pimsleur课程
- 午餐后看视频或读课文
- 睡前Anki词汇复习

### 2. 充分利用Anki
```
每天时间表：
早：新学单词 + 复习旧单词（20分钟）
午：实践练习（20分钟）
晚：睡前复习（10分钟）
```

### 3. 发音很关键
- 使用Forvo对标准发音
- 每天跟读20分钟
- 录音自己的声音与原音对比
- 不要害怕听起来很傻

### 4. 尽早开始说话
- 与自己进行对话
- 想象与他人对话的场景
- 准备好在第3周联系Tandem伙伴

### 5. 追踪你的进度
- 保持学习日志
- 记录困难和突破
- 定期复习较难内容

### 6. 寻找学习同伴
- 加入法语学习社区
- 与朋友分享进度
- 一起学习效果更好
