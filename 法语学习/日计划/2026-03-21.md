## ⏰ 今日学习时间表

### 早晨（08:00-08:40）：字母发音基础 [40分钟]

**任务**：
1. **热身** (5分钟)
   - 播放法语音乐或法语国歌
   - 跟唱，感受语言节奏
   - 推荐：法国国歌《La Marseillaise》

2. **学习26个字母** (25分钟)
   - 打开视频：YouTube "法语26个字母发音" (Linguoni频道)
   - 边看边跟读
   - 特别注意：
     - H是哑音 (h muet)
     - J的发音类似英文中的"z"
     - R要从喉咙发出
     - U和OU的区别

3. **重复练习** (10分钟)
   - 再看一遍视频
   - 这次不看字幕，只听发音
   - 尝试自己发出每个字母的音

**学习资源**：
- YouTube频道：Linguoni - "法语字母发音"
- 时长：约15分钟视频
- URL：youtu.be/[搜索]

**检查点**：
- [ ] 能发音26个字母中的至少20个
- [ ] 理解法语特殊字符（é, è, ê, ë, ù, ç）
- [ ] 感到某种程度的自信（即使不完美）

**反思笔记**：

### 上午（10:00-11:00）：Pimsleur课程1 [60分钟]

**任务**：
1. **准备** (5分钟)
   - 找一个安静的地方
   - 准备笔和纸
   - 关闭所有干扰（手机静音）
   - 戴上耳机

2. **Pimsleur第1课** (45分钟)
   - 课程名称：欢迎和基础问候
   - 内容：
     - "Bonjour" (你好)
     - "Au revoir" (再见)
     - "Je m'appelle..." (我叫...)
     - "Parlez-vous français?" (你会说法语吗?)
   - 方式：跟随应用或音频
   - **重要**：不要急，认真跟读每一句
   - **关键**：学会停顿，自己试着说

3. **笔记和复习** (10分钟)
   - 记下新学的5个关键句子
   - 写下你不理解的地方
   - 重复听最后的评估部分

**学习资源**：
- Pimsleur法语应用 (或音频文件)
- 课程：Lesson 1
- 时长：45分钟
- 成本：可免费试用，或$15/月

**检查点**：
- [ ] 完成整个Pimsleur第1课
- [ ] 理解课程中的主要内容
- [ ] 能重复至少3个新句子
- [ ] 记下5个新词汇

## 💡 今日学习技巧和建议

### 1. 发音很重要，但不用完美
- 今天你的目标是理解，而不是掌握
- 发音会随着练习逐渐改进
- 不要害怕发出奇怪的声音

### 2. 重复是你最好的朋友
- 听一遍不够，至少听3遍
- 跟读10遍才能开始形成记忆
- 这不是浪费时间，这是学习

### 3. 不要完美主义
- 你不需要今天就掌握所有内容
- 这是一个365天的旅程
- 每天进步1%就足够了

### 4. 建立习惯比积累知识更重要
- 这是第一天，重点是建立节奏
- 明天继续这个时间表
- 坚持30天后，就会成为习惯

### 5. 说出来很重要
- 不要只是看和听
- 大声说出来，感受你的舌头和嘴唇的动作
- 这叫"肌肉记忆"
