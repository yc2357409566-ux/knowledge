>
> **开始日期**：2026年3月21日

## 🎯 四个阶段详细说明

### 📍 第一阶段：A1初级（2026年3月-5月）

**目标**：掌握基础发音、日常用语、基础语法

**核心内容**：
- ✅ 法语字母、音素、发音规则
- ✅ 问候、介绍自己、基础问题回答
- ✅ 名词、动词、形容词基础
- ✅ 现在时（être/avoir）
- ✅ 600个高频词汇

**学习资源**：
- Pimsleur法语初级课程
- RFI Savoirs (RFI Savoirs慢速法语)
- 法语发音教学视频

**成果标志**：
- 能够介绍自己和他人
- 能进行简单对话（问候、年龄、职业）
- 能读懂简单菜单和标识
- 能写简单句子

---

### 📍 第二阶段：A2基础（2026年6月-8月）

**目标**：能进行基本生活对话，掌握过去式

**核心内容**：
- ✅ 过去复合时 (Passé Composé)
- ✅ 日常话题对话（购物、餐厅、旅行）
- ✅ 形容词协议、代词系统
- ✅ 常用动词变位扩展
- ✅ 1200个词汇累计

**学习资源**：
- Pimsleur法语中级课程
- TV5Monde新闻视频
- Assimil法语自学教材A2级
- 法国电影/电视剧（带中文字幕）

**成果标志**：
- 能用过去式讲述经历
- 能在餐厅、商店进行对话
- 能理解简单电影情节
- 能写短篇邮件/日记

---

### 📍 第三阶段：B1中级初段（2026年9月-11月）

**目标**：流利交流，理解更复杂内容

**核心内容**：
- ✅ 简未来时 (Futur Simple)

- ✅ 连接词、从句结构
- ✅ 听力理解训练升级

- 能理解新闻报道主旨

## 📚 学习方法论

### 每日学习结构（平均2小时）

```
晨间基础（40分钟）
├─ 听力练习 (20分钟) - Pimsleur或Forvo
├─ 词汇学习 (15分钟) - Anki卡片
└─ 发音练习 (5分钟) - 跟读

午间强化（40分钟）
├─ 语法课程 (25分钟) - 教学视频或教材
└─ 练习题 (15分钟)

晚间应用（40分钟）
├─ 阅读理解 (20分钟) - 新闻或故事
├─ 写作练习 (15分钟) - 日记或邮件
└─ 口语练习 (5分钟) - 自言自语

每周额外（3-4小时）
├─ 电影/电视 (1.5小时)
├─ 阅读深入 (1小时)
├─ 在线交流 (1-1.5小时)
└─ 复习和补漏 (1小时)
```

## 💡 成功建议

1. **坚持是关键** - 每天学习，即使只有15分钟也比不学好
2. **多模式学习** - 不要只读书，要听、说、读、写结合
3. **沉浸式输入** - 多听法语音乐、电视、电台
4. **实际应用** - 尽早开始和真实的人对话
5. **不怕犯错** - 错误是学习的一部分
6. **定期复习** - 使用间隔重复法巩固知识
7. **追踪进度** - 每周回顾自己的进步
8. **保持动力** - 设定小目标，庆祝每个小成就
