## 🎯 必需工具 (优先级1)

### 1. **Pimsleur法语课程** ⭐⭐⭐⭐⭐
**最重要的工具 - 强烈推荐**

| 属性 | 详情 |
|------|------|
| **类型** | 音频互动式课程 |
| **适合阶段** | A1-B1 |
| **时长** | 30分钟/课 × 120课 |
| **成本** | $15/月 or $179/年 or $399/终身 |
| **优点** | 语音交互、间隔重复、效果显著 |
| **缺点** | 需要订阅，不便宜 |
| **URL** | pimsleur.com |

**为什么推荐**：
- Pimsleur是最被证实有效的语言学习系统
- 特别适合初学者
- 强制发音和听力
- 间隔重复算法最优

**如何使用**：
1. 注册账户
2. 下载应用或使用网页版
3. 每天30分钟一课
4. 跟随应用互动
5. 不要跳过或快进

### 2. **Anki** ⭐⭐⭐⭐
**免费词汇学习神器**

| 属性 | 详情 |
|------|------|
| **类型** | 闪卡/间隔重复 |
| **适合阶段** | A1-B1+ |
| **成本** | 免费 (AnkiWeb免费/付费版$25) |
| **优点** | 最强大的记忆工具、免费、可定制 |
| **缺点** | 界面不如商业软件友好 |
| **URL** | ankiweb.net |

**推荐卡片组**：
- "French A1 Vocabulary" (2000张卡)
- "French Common Phrases" (500张)
- "French Verb Conjugations" (1000张)

**最优使用方式**：
```
每天时间表：
- 早：学习20张新卡 (15分钟)
- 午：复习150张卡 (15分钟)
- 晚：复习前几天的卡 (10分钟)
总计：40分钟/天
```

### 3. **RFI Savoirs** ⭐⭐⭐⭐
**免费每日听力和教学**

| 属性 | 详情 |
|------|------|
| **类型** | 每日新闻/教学播客 |
| **适合阶段** | A1-B2 |
| **成本** | 完全免费 |
| **更新频率** | 每天1-2条 |
| **时长** | 5-15分钟/条 |
| **URL** | rfi.fr/cn 或 rfi.fr/fr |

**课程结构**：
- **简单法语** (Français facile) - A1/A2
- **标准法语** (Français courant) - A2/B1
- **法语商务** (Français professionnel) - B1+

**最佳利用**：
1. 早晨听一条新闻
2. 第一遍：只听，不看文字
3. 第二遍：看法语字幕
4. 第三遍：看中文翻译
5. 反复听直到完全理解

## 🎬 电影和电视剧

### 10. **推荐的法国电视剧和电影**

**适合初级（A1-A2）**：
- Extr@ (超级夸张，但简单)
- Madame Claude (电影版，有字幕)

**适合中级（B1）**：
- Lupin (犯罪悬疑剧)
- Engrenages (法庭剧)
- Emily in Paris (虽然有争议但简单)

**适合高级（B1+）**：
- 任何法国电影或电视剧
- 推荐：La Haine, Amélie, Bande de filles等

**观看技巧**：
```
第1周：中文字幕 - 理解情节
第2周：法文字幕 - 同时听和读
第3周：无字幕 - 仅听
第4周：无字幕 - 完全理解
```
