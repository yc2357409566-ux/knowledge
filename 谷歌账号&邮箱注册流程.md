# 谷歌邮箱注册与企业账号激活流程

### 1. 准备工作
* 确保 **Clash Verge** 能正常使用。

### 2. 注册/登录步骤
1.  打开 Chrome 浏览器，点击右上角头像图标。
2.  选择“开启同步功能”或“添加新用户”。
3.  输入公司提供的邮箱地址：
    - 示例：`name@company.com` 或 `name@gmail.com`。
4.  按照系统提示更改初始密码（请务必记录新密码）。

### 3. 安全验证
* 绑定公司指定的备用手机号。
* 开启两步验证（2FA），确保账号安全。

### 4. 浏览器环境同步
> [!important] 核心操作
> 登录成功后，系统会询问“是否开启同步？”，请选择 **“是，我愿意”**。这将自动加载公司预设的书签栏、外贸专用工具以及插件设置。
