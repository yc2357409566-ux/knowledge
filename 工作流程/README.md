## ⚠️ 重要提示

### 🚫 iOS账号特别注意

> **❌ 严禁**：在iPhone设置 → iCloud中登录美区ID
> 
> **✅ 正确**：仅在App Store中登录美区ID
>
> **后果**：登录iCloud可能导致整台设备被远程锁定！

### 🔐 账号安全

- 不要在iCloud中使用美区账号
- 不要启用两步验证用于共享账户
- 定期更新密码（建议3个月一次）
- 保管好邮箱和手机号
